  • The path for the source is https://xvbmc.000webhostapp.com/REPOsitory/
    • ( ...and give it the name "WhateverTheFuck you want" ;-p )
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location,
    select "WhateverTheFuck you want", and install: repository.toetu-1.5.0.zip
  • browse current REPO '.zip-files' @Bitbucket

  • Select the: " (Add-ons) REPOsitory 'et resurrectionis erimus' "
