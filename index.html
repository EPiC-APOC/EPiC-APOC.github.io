ExodusRedux Repository addon

Current Version: 1.4.0

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://xvbmc.000webhostapp.com/REPOsitory/
    ( ...and give it the name "WhateverTheFuck you want" ;-p )
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "WhateverTheFuck you want", and install repository.toetu-1.4.0.zip.
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the: " (Add-ons) REPOsitory 'et resurrectionis erimus' "
  • To use this EPiC repo, see "Addons" — "install from repository" and choose this repo.

Enjoy!
